Test and store your data structures and algorithms code in TypeScript or JavaScript for interview preparation.

## Installation

1. Fork this project and start hacking

2. Install dependencies:

```bash
npm install
```

3. Run testing environment

```bash
npm run test
```

## Data Structures:

- [x] Node
- [x] Stack
